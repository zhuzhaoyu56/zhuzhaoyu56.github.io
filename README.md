# 履带小车基地三维控制台

一个基于 `Vite + Three.js` 的 Web 端履带小车三维控制台，包含模型加载、基地场景、履带动画、手动控制、自动巡航、小地图、环境模拟、传感器可视化和诊断面板。

## 本地运行

```bash
npm install
npm run dev
```

## 生产构建

```bash
npm run build
npm run preview:public
```

## GitHub Pages 部署
